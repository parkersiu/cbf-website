# Coming Soon Static Site

This is a simple static placeholder website.

## Files

- `index.html` — main page content
- `styles.css` — visual styling
- `script.js` — auto-updates the copyright year

## What to edit

Open `index.html` and replace:

- `Your Company`
- `YC`
- `hello@example.com`
- `San Diego, CA`
- The page title and description

## How to use

Upload these files to any static hosting provider, such as:

- Squarespace custom code/page area
- Netlify
- Vercel
- GitHub Pages
- Cloudflare Pages
- Your domain registrar's basic web hosting

For the simplest deployment, upload all files into the website root folder so `index.html` is the homepage.
